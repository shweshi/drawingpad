# sketchpad

Sketch Pad is an easy and clean online sketching tool.

# Features

### Free & Open-Source
- No Ads. Open-Source.

### Offline
- All data are saved in your device. No need to communicate with servers.

### Customizable
- Theme can be customize as per your choice.
